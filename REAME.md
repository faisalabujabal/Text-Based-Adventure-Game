# Text-Based Adventure Game
A school project where we had to program a text-based adventure game using clojure

## Usage

To run the game type you need to have leiningen installed. Type:
> lein run

You can use commands like:
> * north
> * south
> * east
> * west
> * up
> * down
> * status
> * sleep
> * wakeup
> * pick (item)
> * drop (item)
> * fight
> * talk
> * map
> * eat (item)
> * open
